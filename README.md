# E-SPBE Kota Madiun
Antarmuka aplikasi E-SPBE

### Kebutuhan
- Node.js versi v20.9.0

### Cara install
```bash
npm install
```

### mode pengembangan
jalankan perintah berikut di terminal
```bash 
npm run dev
```
buka halaman http://localhost:3000 di browser (chrome/safari/firefox)
